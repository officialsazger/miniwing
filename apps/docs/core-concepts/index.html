Core Concepts

Miniwing is built around token maps, class scanning, and deterministic CSS generation.

Token Sources

  • `tokens.json` is the baseline token dictionary.
  • `miniwing.config.js` can override or extend defaults.
  • The merged result powers utility value lookups.

Class Scanning

  • HTML class attributes are parsed.
  • Known utility patterns are validated.
  • Only discovered classes are emitted as CSS.

Token Override Example

// miniwing.config.js
module.exports = {
  tokens: {
    colors: {
      brand: "#0369a1"
    },
    spacing: {
      18: "4.5rem"
    }
  }
};

Why this model?

It keeps CSS predictable and compact while letting you enforce design values from a single source of truth.

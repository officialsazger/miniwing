Token-driven utility CSS, generated for your project.

Miniwing scans your HTML, resolves classes from tokens, and emits a compact CSS output. No runtime, no heavy framework layer.

Start In 2 Minutes View Real Examples

2core inputs (`tokens.json` + HTML)
1output (`dist/output.css`)

Getting Started

Set up miniwing, run your first build, and wire the output CSS into your docs or app.

1. Install

npm install @sazger/miniwing
# or
yarn add @sazger/miniwing

2. Generate CSS

npx miniwing
# output -> packages/miniwing/dist/output.css

3. Add To HTML

<link rel="stylesheet" href="./packages/miniwing/dist/output.css">

Point your page at the generated CSS file, then use utility classes in markup.

Tip

If class output seems incomplete, verify your scanner paths in `miniwing.config.js`.

4. Example Utility Markup

<button class="bg-blue text-white px-4 py-2 rounded-lg">
  Save Changes
</button>

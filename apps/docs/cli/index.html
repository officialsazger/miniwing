CLI Reference

Command behavior for local development, CI builds, and quick one-off generation.

Main Command

npx miniwing
# or
npm run build

Runs the CLI pipeline: load tokens, scan classes, generate CSS output.

Expected Output

packages/miniwing/dist/output.css

Use this output in docs/app pages via a stylesheet link.

Troubleshooting

  • No CSS update: verify scanner input paths point to your HTML files.
  • Missing utilities: check token keys and utility naming match.
  • Wrong output location: inspect CLI path resolution and config values.
